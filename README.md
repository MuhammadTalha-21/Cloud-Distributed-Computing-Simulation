# ⚡ Distributed Computing System Simulation

### Fault-Tolerant System in C++

---

## 🧠 Overview

This project is a **simulation of a distributed computing system** with built-in **fault tolerance and node crash handling**, developed in **C++**.

It models how multiple computing nodes collaborate to process tasks efficiently while maintaining system stability even when failures occur.

---

## 🚀 Key Features

* 🖥️ Multiple compute nodes with task queues
* ⚖️ Load balancing across active nodes
* 💥 Random node crash simulation
* 🔄 Automatic task redistribution after failure
* 📊 Final execution report of all nodes

## ⚙️ How It Works

1. Tasks are assigned to nodes with the **minimum load**
2. A random node crashes during execution
3. Tasks from the crashed node are redistributed
4. Active nodes continue processing
5. Final report shows completed tasks per node
